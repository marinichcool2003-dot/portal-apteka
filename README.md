# Корпоративный таск-трекер (API Бэкэнд)

Высоконагруженный REST API сервис для автоматизации внутренних процессов и управления задачами. Спроектирован и разработан с использованием актуального корпоративного Java-стека.

## 🚀 Технологический стек
*   **Платформа:** Java 21 / Spring Boot 3.5.7
*   **Безопасность:** Spring Security, JWT (AccessToken + RefreshToken) + генерация защищенных сессий
*   **Базы данных:** PostgreSQL (основное хранилище), Redis (кеширование данных и сессий)
*   **Миграции БД:** Flyway 9.22
*   **Мониторинг и метрики:** Spring Boot Actuator
*   **Документирование:** Springdoc OpenAPI (Swagger UI 2.7)
*   **Инструменты:** Docker / Docker Compose, Maven, Lombok, HikariCP

## ✨ Ключевой функционал и архитектурные решения
1.  **Безопасность (Spring Security & JWT):** Реализована надежная ролевая модель доступа. Аутентификация построена на механизме JWT (с поддержкой времени жизни `expiration` и `remember-me` сессий), пароли хэшируются с помощью `Crypto`.
2.  **Оптимизация производительности (Redis & HikariCP):** Настроено кэширование на базе Redis для снижения нагрузки на реляционную БД. Пул соединений оптимизирован через HikariCP (`maximum-pool-size=30`, `connection-timeout=5000`).
3.  **Строгий контроль данных:** Схемы БД полностью управляются через миграции Flyway. На уровне JPA включена валидация схем данных (`ddl-auto=validate`), что гарантирует стабильность на продакшене.
4.  **Файловый менеджмент:** Реализован изолированный модуль загрузки медиафайлов (аватары, изображения к задачам) с ограничением размера до 10MB и валидацией мультипарт-запросов.
5.  **Наблюдаемость (Observability):** Интегрирован Spring Actuator для мониторинга работоспособности сервиса (Health Checks) и сбора метрик.

## 📦 Контейнеризация и быстрый запуск

Приложение полностью готово к развертыванию в изолированных контейнерах. Все конфиденциальные данные (пароли, ключи) вынесены в переменные окружения.

### Запуск через Docker Compose:
1. Создайте `.env` файл в корне проекта и заполните параметры (база данных, секреты JWT, логины).
2. Запустите инфраструктуру одной командой:
   ```bash
   docker compose up --build
   ```
После старта Docker автоматически поднимет API-сервис, СУБД PostgreSQL (с проверкой `healthcheck`), Redis, а Flyway накатит актуальные миграции базы данных.
